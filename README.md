# RPG-Game

一个基于Unity开发的2D角色扮演游戏（RPG）项目，包含角色系统、战斗机制、场景互动等核心玩法。

## 📋 项目简介

本项目是一款2D像素风格的RPG游戏，旨在打造沉浸式的冒险体验。玩家将扮演骑士角色，探索地图、与敌人战斗、收集道具，并解锁游戏剧情。

- **开发引擎**：Unity 6000.2.9f1
- **美术风格**：2D像素风格
- **核心玩法**：角色移动、战斗系统、场景互动、任务系统

## 🎮 游戏特点

- **丰富的角色动画**：包含骑士的行走、攻击等动作序列
- **多样化的敌人**：如骷髅兵等敌对单位，具备独特的战斗逻辑
- **互动场景元素**：可交互的宝箱、火炬等装饰性与功能性道具
- ** parallax 背景**：实现层次感的滚动背景效果
- **状态系统**：角色属性与战斗状态管理

## 📁 项目结构

- **RPG-Game/**
  - 🗂️ **Assets/**
    - 🎭 **Animations** - 角色与物体动画
    - 🖼️ **Graphics** - 美术资源（精灵图、纹理等）
    - 🎮 **InputSystem** - 输入控制系统  
    - 🎨 **Materials** - 材质与着色器
    - 📦 **Prefab** - 预制体与模板
    - 🏰 **Scenes** - 游戏场景
    - 💻 **Scripts** - 游戏逻辑代码
    - ⚙️ **Settings** - 项目配置设置
    - 🗺️ **Tile Palette** - 瓦片调色板（地图编辑）

## 🚀 快速开始

### 环境要求

- Unity 6000.2.9f1 或兼容版本
- Git（用于版本控制）

### 安装步骤

1. 克隆仓库：
   ```bash
   git clone https://github.com/你的用户名/RPG-Game.git
   ```
2. 打开 Unity Hub，点击Add中「Add project from disk」，选择克隆后的项目文件夹
3. 等待 Unity 导入资源和依赖，完成后即可运行场景，在「Scenes」文件夹中

## 🔧 核心功能实现
- **角色控制**：基于InputSystem实现键盘 / 手柄输入
- **战斗系统**：通过多态和接口完成战斗系统，可拓展性强
- **动画管理**：使用状态机控制开始，改变和结束，Animator组件控制动画播放，减少Animator中的布线
- **滚动背景**：使用视差和背景循环，实现背景运动效果
- **UI界面**：使用UGUI完成血条，背包，技能树等界面
